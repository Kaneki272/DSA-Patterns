# 🧩 DSA Pattern Mastery v2

Stop Memorizing. **Recognize** the **Pattern.**

An interactive, visual, and conceptual guide to mastering Data Structures and Algorithms (DSA) patterns. Every interview problem hides a known pattern. This guide helps you learn to recognize them through visual intuition, problem classification, and code templates.

## ✨ Features

- **Interactive Visualizers**: Step-by-step visualizations of algorithms in action (e.g., Two Pointers, Sliding Window, BFS/DFS, etc.).
- **Pattern Distinguishers**: Clear explanations of when to *apply* a pattern and when to *avoid* it, alongside key signal words you'll see in problem descriptions.
- **Code Templates**: Clean, readable Python 3 implementation templates for every major pattern.
- **LeetCode Integration**: Hand-picked practice problems linked directly to LeetCode, categorized by difficulty.
- **Progress Tracking**: Keep track of the patterns you've learned locally.

## 🛠️ The 15 Core Patterns Covered

1. Two Pointers
2. Sliding Window
3. Prefix Sum
4. HashMap / HashSet
5. Stack (Monotonic)
6. Breadth-First Search (BFS)
7. Depth-First Search (DFS)
8. Backtracking
9. Binary Search
10. Heap / Priority Queue
11. Dynamic Programming
12. Fast & Slow Pointers (Cycle Detection)
13. Merge Intervals
14. Tree DFS / Recursion
15. Topological Sort

## 🚀 Getting Started

Since this is a client-side static web application, no build tools or servers are required! 

Simply clone the repository and open `index.html` in your favorite web browser:

```bash
git clone https://github.com/Kaneki272/DSA-Patterns.git
cd DSA-Patterns
```
*Double-click `index.html` to open it.*

## 💻 Tech Stack

- **HTML5** for structure
- **Vanilla CSS** for modern styling and animations
- **Vanilla JavaScript** natively handling all the complex graph/tree animations and interactive visualizations
